Todo List

This is a demo program for handling a todo list. To exit help, click the "Help" button again. To add a todo list item, enter the name of the todo list item in the text entry box and click "Add Item". To remove a todo list item, click the list item and click "Remove Item". To move a todo list item, click the list item and click either the "Move Up" or "Move Down" button.
